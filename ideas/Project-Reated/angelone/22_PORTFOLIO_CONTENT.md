# Problem Statement

Angel One operated a large-scale public website containing thousands of pages across financial education, product marketing, and customer acquisition.

The engineering challenges included maintaining high performance, supporting rapid marketing campaigns, improving search capabilities, ensuring SEO compliance, and managing an ever-growing content ecosystem.

---

# Solution

Implemented reusable WordPress architecture, optimized frontend performance, automated AMP page generation, integrated Elasticsearch for enterprise search, improved Core Web Vitals, and built reusable components that accelerated future development.
